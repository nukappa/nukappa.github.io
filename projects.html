DVEX: a 3D interactive virtual fly embryo

For the purpose of a research project, I computationally constructed a virtual fly embryo. For each of the 6,000 cells of the embryo, its location in 3-dimensional space is provided, as well as the set of genes that the cell expresses at the given time point. As this was a classification problem, I used machine learning concepts and methods to create the virtual embryo. The end product is hosted in an online interactive database which the user can query to extract information regarding the cells clustering or to predict the spatial expression of a gene on the embryo. DVEX is coded in R and Shiny.

Dropbead: analyzing single-cell sequencing data

I created an R package to easily analyze high-throughput single-cell sequencing data. Its creation was mainly driven by the necessity to quickly explore a new dataset by visualizing its main components and exploring the parameter space. Dropbead was my first package written in R and exposed me to R's inherent differences in defining and treating classes and objects. I ended up with using S4 classes, as it appeared more intuitive for my purposes. Nevertheless, the flexibility and beauty of ggplot have rendered R my favorite environment for performing exploratory data analysis and create visualizations. Dropbead's source code can be found in Github.

A text prediction application

This project was the capstone project of the Data Science Specialization track, offered by the John Hopkins University through the Coursera platform. The task was to create an application that predicts the next word, given a sentence, and wrap it on a Shiny environment. This was my first exposure to NLP concepts. I used Python's libraries to find all 5-grams from the provided data and compute probabilities in order to predict the next word. I considered fall-back scenarios for the cases where no 5-grams were available. The application is hosted on the shinyapps server.
